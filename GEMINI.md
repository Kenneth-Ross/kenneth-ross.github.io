# Project Overview

This is a static personal portfolio website for Kenneth Ross. The site is a single HTML file (`index.html`) that uses CSS for styling and JavaScript for some dynamic behavior like a typewriter effect and scroll animations.

The portfolio showcases:
- An "About Me" section.
- A list of projects with descriptions and images.
- A "Skills & Tools" section.
- A "Contact Me" section with links to LinkedIn and GitHub.

All assets like images and logos are located in the `public/` directory.

# Building and Running

This is a static website, so there is no build process. To view the website, simply open the `index.html` file in a web browser.

# Development Conventions

- All the website's code (HTML, CSS, and JavaScript) is contained within the `index.html` file.
- To make changes or add content, you can directly edit `index.html`.
- Images and other assets should be placed in the `public/` directory and referenced from `index.html`.
